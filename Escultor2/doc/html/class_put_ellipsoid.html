Detailed Description

Represents an ellipsoid that is placed in a Sculptor.

The PutEllipsoid class inherits from the FiguraGeometrica class and provides functionality to place an ellipsoid in a Sculptor object. An ellipsoid is a three-dimensional geometric shape that resembles a stretched or squashed sphere.
