Detailed Description

Represents a sphere that is placed in a Sculptor.

The PutSphere class inherits from the FiguraGeometrica class and provides functionality to place a sphere in a Sculptor object. A sphere is a three-dimensional geometric shape that is perfectly round in shape, like a ball.

This class assumes that the Sculptor class, defined in Sculptor.hpp, is available. The Sculptor class represents a 3D image drawer that uses voxels as building blocks for primitive shapes. The PutSphere class draws the sphere in the Sculptor object.
