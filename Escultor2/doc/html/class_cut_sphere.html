Detailed Description

Represents a sphere that is cut from a Sculptor.

The CutSphere class inherits from the FiguraGeometrica class and provides functionality to cut a sphere from a Sculptor object. A sphere is a three-dimensional geometric shape consisting of all the points that are equidistant from a center point.
